## GET SCRAPPY

This is the beginning of an experimental, collaborative scraper.  A hope with this repository is that ways of ethical scrapping can indeed exist.


### REQUIREMENTS
- Node.js



### To Start

in root directory,

npm install

then

npm install concurrently --save

then

npm install puppeteer

#### To run

npm start
